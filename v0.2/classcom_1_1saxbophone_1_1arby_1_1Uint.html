Detailed Description

Arbitrary-precision unsigned integer type.

Note
std::numeric_limits<Uint> is specialised such that most of the members of that type are implemented to describe the traits of this type.
Exceptions include any members of std::numeric_limits<> which describe a finite number of digits or a maximmum value, neither of which apply to this type as it is unbounded.
Exceptions
std::logic_errormay be thrown from most methods when the result of an operation leaves a Uint object with leading zero digits in its internal representation. Such cases are the result of bugs in this code and should be reported as such.
